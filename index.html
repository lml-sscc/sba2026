步驟 2: DDL 資料表實作

請在 phpMyAdmin 的資料庫中執行以下 SQL DDL 語句來建立基礎資料表：

MySQL DDL (customers, tables, reservations) 
-- 建立顧客資料表
CREATE TABLE customers (
    customer_id INT AUTO_INCREMENT PRIMARY KEY,
    fullname VARCHAR(100) NOT NULL,
    phone VARCHAR(20) NOT NULL UNIQUE
);

-- 建立餐桌/房間資料表
CREATE TABLE tables (
    table_id INT AUTO_INCREMENT PRIMARY KEY,
    table_number VARCHAR(10) NOT NULL UNIQUE,
    seating_capacity INT NOT NULL
);

-- 建立預約紀錄資料表
CREATE TABLE reservations (
    reservation_id INT AUTO_INCREMENT PRIMARY KEY,
    customer_id INT,
    table_id INT,
    reservation_date DATE NOT NULL,
    reservation_time TIME NOT NULL,
    party_size INT NOT NULL,
    FOREIGN KEY (customer_id) REFERENCES customers (customer_id),
    FOREIGN KEY (table_id) REFERENCES tables (table_id)
);

步驟 5: 建立 db.php 共用連線檔

請在專案根目錄下建立 db.php，用於提供統一的 MySQLi 物件導向連線物件：

db.php 
<?php
$host = "localhost";
$username = "root";
$password = "";
$dbname = "hotel_db"; // 你的酒店資料庫名稱

// 使用物件導向 (Object-Oriented) 建立連線
$conn = new mysqli($host, $username, $password, $dbname);

if ($conn->connect_error) {
    die("資料庫連接失敗: " . $conn->connect_error);
}

// 設定編碼，防止中文亂碼
$conn->set_charset("utf8mb4");
?>

步驟 6: 實作動態搜尋網頁 (search.php)

使用 GET 方法 傳遞電話號碼進行跨表 (JOIN) 動態搜尋：

search.php 
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
    <meta charset="UTF-8">
    <title>結構化搜尋預約紀錄</title>
    <style>
        body { font-family: Arial, sans-serif; margin: 40px; background-color: #f9f9f9; }
        .container { max-width: 600px; background: white; padding: 20px; border-radius: 8px; box-shadow: 0 0 10px rgba(0,0,0,0.1); }
        table { width: 100%; border-collapse: collapse; margin-top: 20px; }
        th, td { border: 1px solid #ddd; padding: 8px; text-align: left; }
        th { background-color: #4CAF50; color: white; }
        input[type="text"], input[type="submit"] { width: 100%; padding: 10px; margin: 10px 0; box-sizing: border-box; }
    </style>
</head>
<body>


    <h2>搜尋餐廳訂座紀錄</h2>
    <form method="GET" action="search.php">
        <label>請輸入顧客電話號碼進行搜尋:</label>
        <input type="text" name="search_phone" placeholder="例如:91234567" required>
        <input type="submit" value="開始搜尋">
    </form>

    <?php
    if (isset($_GET['search_phone'])) {
        require_once 'db.php';
        $phone = $_GET['search_phone'];
        $queryResult = fetchReservations($conn, $phone);
        renderTable($queryResult);
        $conn->close();
    }

    function fetchReservations($conn, $phone) {
        $sql = "SELECT c.fullname, r.reservation_date, r.reservation_time, r.party_size, t.table_number 
                FROM reservations r
                JOIN customers c ON r.customer_id = c.customer_id
                JOIN tables t ON r.table_id = t.table_id
                WHERE c.phone = '$phone'";
        return mysqli_query($conn, $sql);
    }

    function renderTable($result) {
        if (mysqli_num_rows($result) > 0) {
            echo "<h3>搜尋結果如下:</h3>";
            echo "<table><tr><th>姓名</th><th>桌號</th><th>日期</th><th>時間</th><th>人數</th></tr>";
            while($row = mysqli_fetch_assoc($result)) {
                echo "<tr>
                        <td>".$row['fullname']."</td>
                        <td>".$row['table_number']."</td>
                        <td>".$row['reservation_date']."</td>
                        <td>".$row['reservation_time']."</td>
                        <td>".$row['party_size']."</td>
                      </tr>";
            }
            echo "</table>";
        } else {
            echo "<p style='color:red;'>找不到與該電話號碼相關的預約紀錄。</p>";
        }
    }
    ?>


</body>
</html>

步驟 7: 實作預約處理網頁 (reserve.php)

使用 POST 方法 提交資料，透過 JavaScript 進行前端基礎格式防錯，並在後端確保交易完整性：

reserve.php 
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
    <meta charset="UTF-8">
    <title>線上訂座</title>
    <style>
        body { font-family: Arial, sans-serif; margin: 40px; background-color: #f4f7f6; }
        .form-box { max-width: 450px; margin: auto; background: #fff; padding: 25px; border-radius: 5px; box-shadow: 0 4px 6px rgba(0,0,0,0.1); }
        input, select { width: 100%; padding: 10px; margin: 10px 0; box-sizing: border-box; }
        input[type="submit"] { background: #18bc9c; color: white; border: none; cursor: pointer; font-size: 1rem; }
    </style>
    <script>
        function validateForm() {
            var name = document.forms["bookingForm"]["fullname"].value;
            var phone = document.forms["bookingForm"]["phone"].value;
            if (name == "" || phone == "") {
                alert("欄位不得為空！");
                return false;
            }
            if (phone.length < 8) {
                alert("請輸入有效的電話號碼（至少 8 位數）");
                return false;
            }
            return true;
        }
    </script>
</head>
<body>


    <h2>餐廳現場訂座系統</h2>
    <form name="bookingForm" method="POST" action="reserve.php" onsubmit="return validateForm()">
        <label>顧客姓名:</label>
        <input type="text" name="fullname" placeholder="例如：陳大文" required>
        
        <label>電話號碼:</label>
        <input type="text" name="phone" placeholder="例如：91234567" required>
        
        <label>預約日期:</label>
        <input type="date" name="res_date" required>
        
        <label>時間:</label>
        <input type="time" name="res_time" required>
        
        <label>選擇餐桌:</label>
        <select name="table_id">
            <option value="1">餐桌 1 (2人座位)</option>
            <option value="3">餐桌 3 (4人座位)</option>
            <option value="6">餐桌 6 (6人座位)</option>
        </select>
        
        <label>用餐人數:</label>
        <input type="number" name="party_size" min="1" max="20" required>
        
        <input type="submit" value="送出預定">
    </form>

    <?php
    if ($_SERVER["REQUEST_METHOD"] == "POST") {
        require_once 'db.php';
        
        $fullname = $_POST['fullname'];
        $phone = $_POST['phone'];
        $res_date = $_POST['res_date'];
        $res_time = $_POST['res_time'];
        $table_id = $_POST['table_id'];
        $party_size = $_POST['party_size'];
        
        // 步驟 A: 檢查顧客是否存在
        $chk_cust = "SELECT customer_id FROM customers WHERE phone = '$phone'";
        $res = mysqli_query($conn, $chk_cust);
        
        if (mysqli_num_rows($res) > 0) {
            $row = mysqli_fetch_assoc($res);
            $customer_id = $row['customer_id'];
        } else {
            $insert_customer = "INSERT INTO customers (fullname, phone) VALUES ('$fullname', '$phone')";
            mysqli_query($conn, $insert_customer);
            $customer_id = $conn->insert_id;
        }
        
        // 步驟 B: 寫入預約
        $insert_res = "INSERT INTO reservations (customer_id, table_id, reservation_date, reservation_time, party_size) 
                       VALUES ($customer_id, $table_id, '$res_date', '$res_time', $party_size)";
        
        if (mysqli_query($conn, $insert_res)) {
            echo "<p style='color:green; font-weight:bold; margin-top:15px;'>🎉 預約成功！資料已寫入資料庫。</p>";
        } else {
            echo "<p style='color:red;'>儲存失敗，錯誤訊息: " . $conn->error . "</p>";
        }
        $conn->close();
    }
    ?>


</body>
</html>
